# CheckButton

## 개요

DayScheduleItem 우측 완료 체크 버튼. Shadow 없음.

## Variants

| Variant | 설명 |
|---|---|
| TRUE / Light | 체크 완료 |
| TRUE / Dark | 체크 완료 |
| FALSE / Light | 미체크 |
| FALSE / Dark | 미체크 |

## 스타일

### Light Mode
- Shadow 없음
- 크기: 24 × 24px
- 완료(`TRUE`): `Light/Primary,CTA Button`
- 미체크(`FALSE`): `Light/Caption,Hint`

### Dark Mode
- Shadow 없음
- 크기: 24 × 24px
- 완료(`TRUE`): `Dark/Primary,CTA Button`
- 미체크(`FALSE`): `Dark/Caption,Hint`
