# NewTravelGenerateButton

## 개요

PlanScreen 빈 상태(PlanScreen-Non)에서만 표시되는 "+ 새 여행 만들기" 버튼.

## Variants

| Variant | 설명 |
|---|---|
| Light | 라이트 모드 |
| Dark | 다크 모드 |

## 스타일

| 속성 | Light | Dark |
|---|---|---|
| Border Radius | `radius-lg` | `radius-lg` |
| Border | `1px solid Light/Primary Hover,Active` | `1px solid Dark/Primary Hover,Active` |
| 배경 | `Light/Primary,CTA Button` | `Dark/Primary,CTA Button` |
| 텍스트 | `heading-sm` / `Light/Surface,Card BG` | `heading-sm` / `Dark/Title,Body Text` |
| Elevation | `Light/elevation-2` | `Dark/elevation-2` |
